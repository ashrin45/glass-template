# Glass Template

Template de base glassmorphism pour projets web.

## Utilisation

1. Cliquer "Use this template" sur GitHub
2. Modifier `index.html` avec votre contenu
3. Les styles sont dans `assets/css/style.css`

## Composants disponibles

- `.section` : bloc de contenu principal
- `.info-box.note` / `.tip` / `.warning` : encadres colores
- `.demo-container` : conteneur d'animation
- `.formula` : affichage de formule
- `.btn` / `.btn-primary` / `.btn-secondary` : boutons
- `.btn-group` : groupe de boutons
- `.slider-group` / `.slider-row` : sliders
- `.nav-bar` : navigation sticky
- `.course-header` : header de page

## Polices

- Plus Jakarta Sans (titres)
- Outfit (corps de texte)
- JetBrains Mono (code, formules)

## Fichiers

```
assets/
  css/style.css      Theme glassmorphism complet
  js/animations.js   Utilitaires canvas (DL object)
  js/bg-mesh.js      Fond anime (blobs + noise)
index.html           Page d'exemple
```
